A simple Android application built with **Java** in **Android Studio**.

## Description
The app follows a short, linear flow across **three screens (activities)**:
1. The user enters basic information about themselves.
2. The user chooses how they want the kitten.
3. A box-opening reveals a kitten image.

## Features
- 3 activities with intent-based navigation
- Basic user input and selection
- Frame-by-frame animation using `AnimationDrawable`

## Tech Stack
- Java  
- Android Studio  
- XML layouts  
- `AnimationDrawable`

## Purpose
This project was created as a simple learning exercise to practice Android fundamentals such as activities, UI design, intents, and basic animations.
